A modern and responsive video portfolio website built using React, JSX, Tailwind CSS, HTML, and CSS. This project showcases my work, skills, and projects in a visually engaging video-based format.

---

## 🚀 Features

- 🎥 Video-based portfolio showcase  
- ⚡ Fast and responsive UI with Tailwind CSS  
- 📱 Fully mobile-friendly design  
- 🎨 Clean and modern interface  
- 🧩 Reusable React components  
- 🌐 Smooth navigation and layout structure  

---

## 🛠️ Tech Stack

- **Frontend:** React.js, JSX  
- **Styling:** Tailwind CSS, CSS3  
- **Markup:** HTML5
- **Tools:** Node.js, npm / yarn
